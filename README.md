#Sprint 4

Vehicle Advertisement Dashboard

Project Description:

This is a web dashboard built with Streamlit. It analyzes a used vehicle advertisements dataset. The graphs allow the users to visually see the relationships between vehicle price, mileage, and condition.

The application is deployed on Render.

Project Objective:

The goal is to practice software development tools and workflows such as:

-Creating a Python project environment
-Using Git and GitHub
-Using EDA on a CSV dataset
-Building a web application with Streamlit
-Creating visualizations
-Deploying a web application to Render
